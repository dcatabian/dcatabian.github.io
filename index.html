What is "Showdown"?

This is a project I did for my web development class at the University of Calgary in the Fall of 2023.

Aside from a Google font, all of the art is mine, and the gameplay is pure Javascript and DOM manipulation.

I was kind of proud of how it turned out, so here it is as my showcase website, modified slightly with this explanation.

INSTRUCTIONS

This two-player game is about which of you have the quicker draw!

When the game begins, each player is given the set of possible keys they will have to press:

Player 1 will get (at most) these keys

Player 2 will get (at most) these keys

When both players confirm, get ready. Because after a random amount of time the game will go

DRAW

and the game will display the key each player has to press.
Whoever presses their designated key first wins!

Be careful though, if you press the wrong key you will miss, and your opponent might be able to get a free shot!

if both players miss, you'll move closer and you'll get a smaller set of possible keys.
